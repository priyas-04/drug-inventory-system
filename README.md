# Drug Inventory and Supply Chain Management System

## 📌 Overview

The Drug Inventory and Supply Chain Management System is a full-stack web application designed to help pharmacies, hospitals, and warehouses efficiently manage medicines from procurement to distribution. It ensures accurate stock tracking, supplier management, and secure role-based access.

## 🛠 Tech Stack 

* **Backend:** Java, Spring Boot, Spring Data JPA, Hibernate
* **Frontend:** React (Vite), JavaScript, CSS
* **Database:** MySQL
* **Tools:** Maven, Git, Postman

## 🚀 Features 

* Role-based authentication (Admin, Pharmacist, Supplier)
* Drug inventory management (Add, Update, Delete)
* Real-time stock tracking with low stock alerts
* Expiry date tracking and validation
* Supplier and purchase order management
* Stock movement tracking (IN / OUT)
* Reports dashboard for analytics

## ▶️ How to Run the Project 

### 🔹 Backend (Spring Boot)

```bash
mvn spring-boot:run
```

### 🔹 Frontend (React)

```bash
cd frontend
npm install
npm run dev
```
